## Module Summaries

### Task 1 – Digital Literacy Awareness Infographic (Module 1)
Created a one-page infographic using **Canva** covering five key areas of digital literacy: what digital literacy is, useful digital tools for students, internet safety, professional online presence, and email etiquette. The infographic is saved in the `task-1-presentation/` folder.

### Task 2 – Student Digital Portfolio (Module 2)
Set up professional profiles on **GitHub** and **LinkedIn**. GitHub profile includes a README with my name, branch, year, and learning goals. LinkedIn profile includes my education details at VIT Bhopal University (B.Tech CSE, 2025–2029). Screenshots are saved in `task-2-portfolio/`.

### Task 3 – Coding & Collaboration Platforms (Module 3)
Completed the **"Solve Me First"** challenge on HackerRank using Python 3 (score: 1.0 / Accepted). Also created a **Digital Literacy Awareness Quiz** using Google Forms with 5 questions. Screenshots and response sheet are in `task-3-platforms/`.

> 🔗 [Google Form Link – Digital Literacy Awareness Quiz](#) *https://docs.google.com/forms/d/e/1FAIpQLSelzFpn4bzQf_GhLyBDBccZBoXWQmqKENLr8BvrVp_CeKysqQ/viewform?usp=publish-editor*

### Task 4 – Email Etiquette (Module 4)
Drafted two professional emails: one requesting an assignment extension and one applying for a summer internship. Also created a Social Media Do's and Don'ts checklist. Files are in `task-4-email-etiquette/`.

### Task 5 – Cybercrime Awareness (Module 5)
Researched and wrote a case study on **UPI / Online Payment Fraud** — one of the most prevalent cybercrimes targeting college students in India. Also created a 10-point prevention checklist including cybercrime.gov.in and Helpline 1930. Files are in `task-5-cybercrime/`.
